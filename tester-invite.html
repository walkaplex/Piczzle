Install note

Android may warn that the app came from an unknown source. That is expected for this private test build. Only install it if you are comfortable testing an early app.

How to install

  1. Download the Piczzle test build from this page.
  2. Unzip the downloaded file and open the APK on your Android phone.
  3. If Android asks, allow installation from the app you used to open the file.
  4. Confirm install, then open Piczzle.

What to try

  • Create a puzzle from a demo image.
  • Create a puzzle from one of your own photos if you are comfortable.
  • Try crop and zoom before creating the puzzle.
  • Try Hint, Restart, Solve, Save Image, and Share Puzzle.
  • Open a shared puzzle link if one is sent to you.

What to send back

  • Your Android phone model.
  • Your Android version if you know it.
  • Screenshots or a screen recording if anything looks wrong.
  • What you tapped right before any issue.
  • Anything confusing, broken, slow, or awkward.

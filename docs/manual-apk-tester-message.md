# Manual APK Tester Message

Use this when sending the private debug APK directly to trusted Android testers.

## Short Message

Hi! I am testing an early Android build of Piczzle, a photo puzzle app.

This is a private test build, not a Play Store release yet. Android may warn you that the app came from an unknown source. Only install it if you are comfortable testing an early build.

Please try:

- Open Piczzle.
- Create a puzzle from one of the demo images.
- Create a puzzle from one of your own photos if you are comfortable.
- Try crop/zoom, Hint, Restart, Solve, Save Image, and Share Puzzle.
- Open a shared puzzle link if one is sent to you.

Please send back:

- Your Android phone model.
- Your Android version if you know it.
- Screenshots or a screen recording if anything looks wrong.
- What you tapped right before any issue.
- Anything confusing or awkward.

Privacy note:

https://walkaplex.github.io/Piczzle/privacy.html

## Files To Send

- `piczzle-debug-20260602-1126.apk`
- `piczzle-debug-20260602-1126.txt`
- Optional: this message copied into the email/text thread.

## Tester Install Notes

The tester usually needs to:

1. Download the APK on their Android phone.
2. Open the downloaded APK.
3. Allow installation from that app if Android asks.
4. Confirm install.
5. Open Piczzle.

Different Android phones word the warning differently. That is expected for direct APK testing.

## Do Not Send Publicly

Only send this APK to trusted testers. It is not a public release and should not be posted publicly.

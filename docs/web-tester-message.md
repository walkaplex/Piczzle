# Web Tester Message

Use this when inviting iPhone, iPad, desktop, or browser-based testers.

## Short Message

Hi! I am testing Piczzle, a photo puzzle app.

If you are on iPhone or iPad, please open this in Safari:

https://walkaplex.github.io/Piczzle/web-tester-invite.html

For an app-like feel on iPhone or iPad:

1. Open the invite in Safari.
2. Tap Share.
3. Choose Add to Home Screen, then tap Add.

Please try:

- Open Piczzle.
- Create a puzzle from one of the demo images.
- Create a puzzle from one of your own photos if you are comfortable.
- Try crop/zoom, Save Image, and Share Puzzle.
- Open a shared puzzle link if one is sent to you.

Please send back:

- Your device model.
- Your browser.
- Screenshots or a screen recording if anything looks wrong.
- Anything confusing, broken, slow, or awkward.

Privacy note:

https://walkaplex.github.io/Piczzle/privacy.html

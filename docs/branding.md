# Piczzle Branding

## Core Idea

Piczzle should feel like a playful photo puzzle app. The brand mark should read as a puzzle piece immediately, even at small launcher-icon size.

## Icon Direction

- Use one clear jigsaw puzzle piece as the primary symbol.
- Use the premium glossy 3D icon as the main production direction.
- Fill the piece with four bright photo-like color tiles: green, cyan, yellow, and violet.
- Keep the app background dark navy so the piece pops.
- Avoid face-like details, mascot shapes, alien/creature silhouettes, or abstract marks that do not immediately say puzzle.
- Use the same mark for launcher, splash screen, web favicon, and in-app header logo.

## Visual Tone

- Bright, approachable, and game-like.
- Polished enough for an app store listing.
- Simple enough to recognize in the Android launcher or iOS home screen.
- Dimensional and glossy is welcome as long as the jigsaw shape stays immediately clear.

## Splash Direction

- Use a dark navy background.
- Center the same puzzle-piece mark.
- Avoid extra text so the launch screen stays clean and fast.

## Current Brand Colors

- Navy background: `#0B1020`
- Panel navy: `#151D31`
- Green tile: `#32E6A3`
- Cyan tile: `#27CFFF`
- Yellow tile: `#FFD45A`
- Violet tile: `#7D5CFF`
